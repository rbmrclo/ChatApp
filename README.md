# ChatApp

  A Sails.js + Backbone.js application
  
### Installation

  Make sure you have sails.js installed. If not, you can do it by:
  
    $ sudo npm -g install sails
    
  Then install all the dependencies by:
  
    $ npm install
    
### Start It Up

  Lets do this!
  
    $ sails lift
